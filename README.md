# SelfX
This is a live online application that supports multiple languages, global time zones, and works across PC, tablet, and mobile devices.
The URL is selfx.io

这是一个在线运行的应用程序，支持多语言、全球时区，并可在电脑、平板和手机等多种设备上使用。
网址是 selfx.io。

これは稼働中のオンラインアプリケーションで、多言語対応・グローバルタイムゾーン対応、そして PC・タブレット・スマートフォンなど複数のデバイスで利用できます。
URL は selfx.io です。
